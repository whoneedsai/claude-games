# Bjossi's Claude Games

A collection of classic games built with Claude, using vanilla HTML, CSS, and JavaScript.

## Games

| Game | Description |
|------|-------------|
| [Kotra](games/kotra/) | Backgammon (the Icelandic name) — single player vs AI |

## How to play

Open `index.html` in any modern browser. No build tools or dependencies required.
